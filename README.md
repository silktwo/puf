# PUF Site

Static multi-page site for /PUF.

## Local Preview

```bash
npm run dev
```

Open `http://127.0.0.1:5174/`.

## Deploy To Vercel Through GitHub

1. Create a new GitHub repository.
2. Push this project to the repository.
3. In Vercel, choose **Add New Project** and import the GitHub repo.
4. Use framework preset **Other**.
5. Leave build command empty.
6. Leave output directory empty.
7. Deploy.

The full PDF catalog in `assets/Catalog/` is intentionally ignored and should not be committed or deployed.
